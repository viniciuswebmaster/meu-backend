# PHP Backend API

Simple backend API built with pure PHP, following a layered architecture.

## Features

- Basic routing
- Controller / Service / Repository structure
- MySQL integration
- JSON responses

## Tech Stack

- PHP
- MySQL
- Git

## Endpoints

- `GET /?route=items` → list items
- `GET /?route=create` → create item (mock)

## How to run

```bash
php -S localhost:8000
